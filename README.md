# 이베이 쇼피 단가 계산기

상품 원가와 목표 마진을 입력하면 이베이 권장가와 쇼피 국가별 권장 리스팅가를 계산하는 웹앱입니다.

## 사용 방법

1. `index.html`을 브라우저로 엽니다.
2. 상품 원가, 국내 배송비, 포장비, 목표 마진, 무게를 입력합니다.
3. 목표 마진은 `원화` 또는 `원가 대비 %` 중 하나로 선택합니다.
4. 이베이 권장가와 쇼피 국가별 권장가를 확인합니다.
5. 실제 계정 수수료가 다르면 `이베이 설정` 또는 `쇼피 설정`을 열어 수정합니다.
6. 나중에 다시 찾을 상품은 `전체 상품 저장`을 누릅니다.

입력값은 브라우저에 자동 저장됩니다. 다시 기본 샘플로 돌리고 싶으면 `샘플로 되돌리기` 버튼을 누르면 됩니다.

## 저장과 검색

- 저장한 상품은 `저장한 상품 검색` 표에 남습니다.
- 상품명, 국가, 추천 채널로 검색할 수 있습니다.
- `불러오기`를 누르면 해당 상품의 원가, 배송비, 무게, 국가, 목표 마진이 다시 입력됩니다.
- 저장 내용은 이 PC의 브라우저 안에 저장되며 외부로 전송하지 않습니다.

## 엑셀에서 옮긴 기준

- 이베이: `단가 계산` 시트의 원가/수수료/마진 흐름을 우선 반영
- 쇼피: 싱가폴, 대만, 브라질, 태국, 말레이시아, 필리핀, 베트남, 멕시코 국가별 계산과 `배송_국가` 시트 및 멕시코 공식 요율표의 배송비/Hidden Fee를 반영
- `아이디 비번`, `계정` 같은 민감정보 시트는 웹앱으로 옮기지 않음

## 계산 기준

- 이베이는 해외 배송비를 상품 마진 계산에서 제외합니다.
- 쇼피는 국가별 배송비/Hidden Fee를 포함해 리스팅가를 계산합니다.
- 목표 마진은 원화 또는 원가 대비 % 기준입니다. % 방식은 상품 원가 기준으로 원화 마진을 계산합니다.
- 이베이 USD 환율은 1,335원 고정입니다.
- 쇼피 환율은 하나은행 최초 고시 매매기준율의 92%를 기본값으로 사용합니다.
- 이베이는 USD만 표시하고, 쇼피는 국가별 통화로 표시합니다.
- 쇼피 판매 수수료와 거래 수수료는 교육 허브 10624번 글의 VAT 포함 최신본 기준을 기본값으로 사용합니다.
- 쇼피 판매 수수료는 SLS 방식 기준입니다. 설정값이 이상하면 `쇼피 설정`의 `SLS 공식 수수료 적용` 버튼으로 공식 기본값을 다시 넣을 수 있습니다.
- 거래 수수료는 상품가뿐 아니라 구매자가 배송비로 지불한 금액까지 포함한 주문 금액 기준으로 계산합니다.
- 인출 수수료는 리스팅가 전체가 아니라 쇼피 수수료 차감 후 정산 받은 대금 기준으로 계산합니다.
- FSP/서비스 수수료는 모든 국가 FSP 가입 가능성을 고려해 기본 4%로 계산하며, 실제 계정 조건에 맞게 수정할 수 있습니다.
- 쇼피 일반 SLS 주문의 수입 관부가세는 쇼피 자동 마크업 또는 구매자 부담으로 보고 판매자 마진에서 제외합니다.
- 통관 완료 후 반품/배송 실패처럼 판매자가 수입 관부가세를 부담할 수 있는 예외는 `쇼피 설정`에서 `예외: 판매자 부담`으로 바꾸면 국가별 추정 관부가세를 가격에 포함합니다.

## 리서치 반영

- 이베이 공식 설명서는 최종 가치 수수료가 상품가뿐 아니라 구매자가 결제한 배송비, 세금, 기타 비용까지 포함한 총액 기준으로 계산될 수 있다고 안내합니다. 그래서 앱은 사용자의 업무 기준대로 배송비를 마진에서 제외하되, 실제 정산과 차이가 날 수 있음을 결과 문구에 표시합니다.
- 쇼피 공식/도움말 문서는 국가별 수수료, 거래 수수료, 배송 방식이 시장마다 다르다고 안내합니다. 앱은 쇼피 엑셀의 국가별 배송비/Hidden Fee 표를 우선 기준으로 사용하고, 수수료와 환율은 화면에서 직접 수정할 수 있게 했습니다.
- 쇼피 교육 허브 10624번 글은 판매 수수료, 거래 수수료, 페이오니아 인출 수수료, 서비스 수수료를 구분합니다. 앱도 이 구분을 따라 판매 수수료와 거래 수수료를 국가별로 두고, FSP/CCB/Paid CS 등 서비스 수수료는 별도 입력값으로 둡니다.
- 멕시코는 Shopee Korea 마켓별 해외 배송비 안내의 Google Sheet `KR-MX SLS Rate Table & Price tool_2026` 중 `NEW MX Rate Table (2026.07.01~)`와 `MX Price Tool`을 함께 대조해 사용합니다. 구매자 부담 배송비는 40 MXN이며, 앱 원가에 넣는 seller hidden fee는 Price Tool 기준 hidden fee 열입니다.
- 쇼피 관세 안내 문서 기준으로 태국 SLS는 2026년부터 쇼피가 관부가세 포함 가격을 자동 마크업하며, 멕시코도 플랫폼이 수입 관세율을 자동 반영합니다. 그래서 앱은 일반 주문의 관부가세를 판매자 고정비로 넣지 않고, 통관 완료 후 반품/배송 실패 등 판매자 부담 예외만 별도 모드로 계산합니다.
- 외부 쇼피 수수료 계산기들은 보통 `상품 원가 -> 플랫폼 수수료 -> 배송/프로그램 비용 -> 목표 이익 -> 권장 리스팅가` 순서로 계산합니다. 현재 앱도 이 흐름에 맞춰 한 상품 입력 후 이베이와 쇼피 국가별 가격표를 바로 보여줍니다.
# Railway 없이 사용하는 방법

이 계산기는 서버 없이도 사용할 수 있는 정적 웹앱입니다.

- PC에서 바로 열기: `open-calculator.bat` 더블클릭
- 로컬 웹주소로 열기: `start-local-web.bat` 더블클릭 후 `http://127.0.0.1:8765/index.html`
- 무료 웹 배포: `web-release` 폴더 안의 파일만 GitHub Pages, Netlify, Vercel 등에 업로드
- 현재 발행 주소: https://mizukikorea1005.github.io/listing-price-calculator/

자세한 순서는 `FREE-WEB-HOSTING.md`를 확인하세요.
